image2fusebeads
===============

Convert your image to a fuse beads pattern

Requirements
------------

* bash
* imagemagick

For Ubuntu:
```bash
sudo apt-get install imagemagick
```

Usage
-----

```bash
bin/image2fusebeads [IMAGE1 [IMAGE2 ...]]
```

Without any arguments, the example files get processed:

```bash
bin/image2fusebeads 
```
